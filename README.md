# 粒子土星环项目使用说明

项目是一个基于摄像头手势控制的粒子土星环展示页。用户可以通过张开手掌、左右滑动、捏合等动作来触发照片墙和照片切换。

## 1. 页面上有哪些按钮

页面上主要有这些可见按钮和控件：

- `开启星际之旅`
- `全屏沉浸`
- `上传图片`
- `暂停捕捉`
- `摄像头选择下拉框`
- `控制面板（GUI）`

## 2. 如何启动项目

浏览器打开：

```text
https://photo.witandpep.top/
```

## 3. 如何启动手势捕捉

手势捕捉不是打开页面就自动完全开始的，需要先做下面两步：

1. 打开页面
2. 点击页面中央的 `开启星际之旅`

点击后会发生这些事：

- 启动音频
- 初始化摄像头
- 请求浏览器摄像头权限
- 开始手势识别

如果浏览器弹出摄像头权限窗口，要点“允许”。

## 4. 如何暂停手势捕捉

有两种方式：

### 方法 A：按空格键

- 按一次空格：暂停手势识别
- 再按一次空格：恢复手势识别

页面下方提示里也写着：

```text
空格键暂停 / 恢复手势
```

### 方法 B：点击底部的 `暂停捕捉` 按钮

这个按钮控制的是摄像头视频捕捉本身：

- 显示 `暂停捕捉` 时，点一下会暂停视频流
- 暂停后按钮会变成 `恢复捕捉`
- 再点一次会恢复视频流

简单理解：

- `空格键`：暂停手势识别逻辑
- `暂停捕捉` 按钮：暂停摄像头视频输入

## 5. 如何上传照片

页面右侧有 `上传图片` 按钮。

操作方式：

1. 点击 `上传图片`
2. 在系统文件选择器里选中一张或多张图片
3. 上传完成后，页面会显示 `已加载 N 张图片`

说明：

- 支持一次选多张
- 上传成功后，这些图片会进入后续的照片墙和特写切换流程

## 6. 如何进入照片模式

当图片已经上传后：

1. 人站在摄像头前
2. 张开手掌

这时会触发：

- 粒子爆开
- 照片墙出现

如果没有上传图片，系统会显示治愈文案，而不是照片。

## 7. 如何切换照片

照片切换分两个阶段：

### 阶段 A：从照片墙进入特写

张开手掌后，系统会先停留在照片墙状态。

这时只要：

- 手向左滑
- 或手向右滑

系统就会把一张照片从照片墙里拉到前景，进入特写模式。

### 阶段 B：在特写照片之间切换

进入特写后，继续左右滑动手掌即可切换上一张/下一张照片。

简单记忆：

- 第一次滑动：从照片墙聚焦到一张图
- 之后再滑动：切换前景照片

## 8. 如何退出照片模式

在照片墙或照片特写状态下，做 `捏合` 手势即可退出。

退出时会发生：

- 照片消失
- 粒子收回
- 回到初始土星环状态

## 9. 页面上每个按钮的作用

### `开启星际之旅`

作用：

- 启动音频
- 触发摄像头初始化
- 让浏览器开始进入可交互状态

什么时候用：

- 第一次打开页面时必须先点

### `全屏沉浸`

作用：

- 进入浏览器全屏模式

再次点击后：

- 退出全屏

什么时候用：

- 现场展示或投屏时很有用

### `上传图片`

作用：

- 导入本地图片到画廊

上传后：

- 页面会提示已经加载了多少张图片

### `暂停捕捉`

作用：

- 暂停摄像头视频输入

暂停后按钮会切成：

- `恢复捕捉`

### `摄像头选择下拉框`

作用：

- 切换当前使用的摄像头设备

适合场景：

- 笔记本前后摄像头切换
- 外接摄像头切换

## 10. GUI 控制面板里每一组按钮是做什么的

右上角的 GUI 默认是折叠的，点击分组名称可以展开。

### `外观`

- `粒子大小`：调整土星环粒子的大小
- `辉光强度`：调整粒子的发光感

适合：

- 调整画面细腻程度和“发光氛围”

### `光环设置`

- `内环宽度`：调整内侧粒子环的宽度
- `外环宽度`：调整外侧粒子环的宽度
- `环缝间距`：调整内外环之间的空隙
- `光环厚度`：调整环带整体厚度

适合：

- 改造土星环的整体造型

### `运动控制`

- `粒子自转`：开关粒子环自身旋转
- `自转速度`：调节旋转快慢
- `旋转方向`：切换顺时针 / 逆时针
- `轴倾角`：调整土星环倾斜角度

适合：

- 调整动态感和构图姿态

### `镜头控制`

- `距离 (Zoom)`：控制镜头远近
- `高度 (Elev)`：控制镜头上下位置
- `自动环绕`：开关镜头自动绕场景旋转

适合：

- 快速找到更适合展示的视角

### `系统参数`

- `粒子总数`：调整粒子数量
- `背景繁星`：调整背景星星数量
- `星空亮度`：调整背景星空的亮度

适合：

- 在性能和视觉效果之间做平衡

注意：

- 粒子数量越多，机器压力越大

### `音频设置`

- `主音量`：调整整体音量

适合：

- 现场展示时快速把音乐调大或调小

### `照片画廊`

- `特写尺寸`：调整前景大图尺寸
- `照片墙半径`：调整照片墙围绕场景的半径
- `缩略图大小`：调整照片墙中每张小图大小
- `墙透明度`：调整照片墙透明度
- `切换速度`：调整照片切换动画速度
- `滑动灵敏度 ↔`：调整左右滑动触发切换的灵敏度
- `切换间隔 (ms)`：控制两次切换之间的最小间隔

适合：

- 优化照片墙的观感和手势切换体验

### `摄像头预览`

- `显示预览`：显示 / 隐藏右下角摄像头画中画
- `预览尺寸`：调整右下角画中画大小

适合：

- 调试手势识别是否稳定

### `配置存储`

- `💾 保存当前配置`：保存当前 GUI 参数，下次刷新页面仍然保留
- `↺ 重置默认`：清空保存配置并恢复默认参数

适合：

- 调好展示参数后固定下来

## 11. 页面上的状态说明

底部状态条会提示当前状态，常见有：

- `初始化...`
- `手势待命`
- `手势追踪中...`
- `未检测到手部`
- `手势输入已暂停`
- `捕捉已暂停`

简单理解：

- `手势输入已暂停`：一般是按了空格
- `捕捉已暂停`：一般是点了底部的 `暂停捕捉`

## 12. 推荐最短操作流程

如果只是快速演示，按这个顺序即可：

1. 打开页面
2. 点击 `开启星际之旅`
3. 允许摄像头权限
4. 点击 `上传图片`
5. 面向摄像头张开手掌
6. 左右滑动切换照片
7. 捏合退出照片模式
8. 需要暂停识别时按空格

## 13. 常见问题

### 页面打开了，但手势没反应

先检查：

- 有没有点 `开启星际之旅`
- 浏览器有没有允许摄像头
- 底部状态是否显示 `手势输入已暂停`
- 是否误点了 `暂停捕捉`

### 看得到画面，但切换照片不顺

可以去 GUI 里调整：

- `滑动灵敏度 ↔`
- `切换间隔 (ms)`
- `切换速度`

### 想让展示更稳定

建议：

- 光线充足
- 摄像头正对手部
- 背景尽量简洁
- 不要让手离镜头太近

## 14. Fork 后如何本地运行

如果同事需要基于源码修改，可以先 fork 这个仓库，再本地运行：

```bash
git clone <你的-fork-地址>
cd 粒子
npm install
npm run dev
```

本地开发地址默认是：

```text
http://localhost:5173
```
